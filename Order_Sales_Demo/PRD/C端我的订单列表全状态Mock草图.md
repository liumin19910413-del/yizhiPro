# C端我的订单列表全状态 Mock 草图

> 用途：用于确认 C 端「我的订单」列表页在不同 Tab、不同订单业务形态、不同售后状态下的信息区展示。
>
> 口径：列表页以订单信息展示和普通订单操作为主；售后发起动作进入订单详情页处理。售后 Tab 展示售后订单，并提供售后详情入口。

---

## 1. 页面 Tab 与通用卡片结构

### 1.1 顶部 Tab

```text
+--------------------------------------------------+
|  <                         我的订单        ... o |
+--------------------------------------------------+
|  全部  待付款  待成团  待发货  待收货  待自提  🔍 |
|  待评价  已完成  售后                             |
+--------------------------------------------------+
```

### 1.2 普通订单卡片骨架

```text
+--------------------------------------------------+
|  订单编号  202606070001                 订单状态 |
|  ..............................................  |
|  [图片]  商品/套餐/预定金/卡名称                 |
|          SKU规格信息                             |
|                                                  |
|          ￥98                              x1    |
|                                                  |
|                         共1件，实付款 ￥98       |
|                                      [普通按钮]  |
+--------------------------------------------------+
```

普通订单卡片字段规则：

| 区域 | 展示内容 |
| --- | --- |
| 订单头部 | 左侧订单编号，右侧订单状态 |
| 图片 | 商品单显示商品主图；套餐单显示套餐图片；预定金单显示预定金图片；开充卡单显示储值卡图片 |
| 名称 | 商品名、套餐名、预定金名称、`开卡 卡名字` 或 `充值 卡名字` |
| SKU | 商品/套餐/预定金展示 SKU 规格；开充卡展示 `充值** 赠送**` |
| 单价与数量 | 左侧展示单价或积分/现金组合，右侧展示数量 |
| 汇总 | 待付款展示待付款金额；已支付展示实付款；纯积分兑换展示实付积分；积分加钱展示现金加积分 |
| 按钮 | 只展示普通订单按钮，不展示售后发起类按钮 |

### 1.3 售后订单卡片骨架

```text
+--------------------------------------------------+
|  订单编号  202606070900                 售后状态 |
|  ..............................................  |
|  [图片]  退款商品名称                             |
|          SKU规格信息                             |
|                                                  |
|          原价 ￥98          退1件，退款 ￥98      |
|                                                  |
|              共1件，申请退款 ￥98，实际退款 ￥98  |
|                                      [售后详情]  |
+--------------------------------------------------+
```

售后订单卡片字段规则：

| 区域 | 展示内容 |
| --- | --- |
| 订单头部 | 左侧订单编号，右侧售后状态 |
| 商品信息 | 展示本次退款商品，不展示整单全部商品 |
| 商品金额 | 展示原商品单价、退款数量、该商品本次退款金额 |
| 汇总 | 未完成展示申请退款金额；完成态展示申请退款金额和实际退款金额 |
| 按钮 | 统一展示 `[售后详情]` |

## 11. 状态归属规则

| Tab | 进入条件 | 备注 |
| --- | --- | --- |
| 全部 | 所有普通订单与售后订单 | 混合展示 |
| 待付款 | 未支付订单 | 只有该状态展示删除订单 |
| 待成团 | 未达成成团条件的订单 | 成团后按业务形态进入后续状态 |
| 待发货 | 商品发货订单支付完成或成团后等待发货 | 只展示提醒发货 |
| 待收货 | 商品发货订单已发货等待收货 | 展示查看物流、确认收货 |
| 待自提 | 商品自提订单等待提货 | 展示查看自提码 |
| 待评价 | 商品发货确认收货后、商品自提提货后、套餐购买完成后 | 预定金和开充卡不进入该状态 |
| 已完成 | 商品/套餐评价完成；预定金、开卡、充值支付完成 | 无按钮 |
| 售后 | 所有售后订单 | 展示售后详情 |

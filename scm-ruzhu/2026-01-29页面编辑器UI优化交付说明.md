## 重要提示：清除浏览器缓存

### ⚠️ 必须清除缓存才能看到更新

由于浏览器会缓存JavaScript文件，您需要清除缓存才能看到最新的更新内容。

### 快速清除方法（推荐）

**Mac系统 - 强制刷新**：
```
Chrome/Edge: Command + Shift + R
Safari: Command + Option + R
Firefox: Command + Shift + R
```

**或者使用开发者工具**：
1. 打开开发者工具：`F12` 或 `Command + Option + I`
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

### 验证更新是否生效

访问版本检测页面：
```
http://localhost:8000/dist/test-editor-version.html
```

此页面会自动检测所有新增的函数是否存在，如果检测失败会提示清除缓存。

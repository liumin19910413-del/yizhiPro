# 统一入口功能 - 交付说明

## 📦 交付内容

### 新增功能：统一入口页面

根据您的需求，我创建了一个统一的入口页面，让您可以在同一个浏览器页面中快速切换MG管理后台和Lite商户后台，无需打开多个标签页。

## ✨ 核心特性

### 1. 一个页面，两个系统

```
之前：需要打开两个浏览器标签
┌─────────────┐  ┌─────────────┐
│ MG管理后台   │  │ Lite商户后台 │
└─────────────┘  └─────────────┘

现在：一个页面搞定
┌──────────────────────────────┐
│ 统一入口（包含MG和Lite）       │
│ [MG管理后台] [Lite商户后台]   │
└──────────────────────────────┘
```

### 2. 顶部快速切换

```
┌─────────────────────────────────────────────────────┐
│ 🏪 店商供应链系统  [MG管理后台] [Lite商户后台]  🔄 🗑️ │
└─────────────────────────────────────────────────────┘
```

- 点击按钮即可切换系统
- 当前系统高亮显示
- 切换流畅无延迟

### 3. 便捷操作

| 功能 | 说明 | 快捷方式 |
|------|------|----------|
| 切换系统 | 在MG和Lite之间切换 | 点击顶部按钮 |
| 刷新系统 | 重新加载当前系统 | 点击🔄按钮 |
| 重置数据 | 恢复初始测试数据 | 点击🗑️按钮 |
| 记忆功能 | 自动记住上次访问 | 自动保存 |

## 🚀 使用方法

### 快速开始

```bash
# 直接打开
open prototype/index.html

# 或使用服务器
cd prototype
./start-server.sh
# 访问 http://localhost:8000/index.html
```

### 操作流程

1. **首次打开**
   - 看到欢迎页面
   - 点击卡片或顶部按钮选择系统

2. **切换系统**
   - 点击顶部的「MG管理后台」或「Lite商户后台」按钮
   - 系统立即切换，无需等待

3. **刷新系统**
   - 点击顶部的「🔄 刷新」按钮
   - 当前系统重新加载

4. **重置数据**
   - 点击顶部的「🗑️ 重置数据」按钮
   - 确认后数据恢复初始状态
   - 点击刷新查看效果

## 🎯 使用场景

### 场景1：测试联动功能

**需求：** 测试Lite端申请 → MG端审核的完整流程

**传统方式：**
1. 打开Lite系统标签页

5. 查看状态

**优势：** 操作更流畅，不会找错标签页

### 场景2：对比数据

**需求：** 对比两个系统中同一订单的数据

**传统方式：**
1. 在MG系统查看订单
2. 记住订单号
3. 切换标签页到Lite系统
4. 搜索订单
5. 对比数据

**统一入口方式：**
1. 在MG系统查看订单
2. 点击顶部切换到Lite系统
3. 搜索订单
4. 对比数据

**优势：** 切换更快，不会迷失在多个标签页中

### 场景3：演示展示

**需求：** 向他人演示系统功能

**传统方式：**
- 需要在多个标签页间切换

- 演示不够流畅

**统一入口方式：**
- 一个页面完成所有演示

- 演示更加专业

## 🔧 技术实现

### 架构设计

```
┌─────────────────────────────────────┐
│         统一入口页面                 │
│  ┌─────────────────────────────┐   │
│  │      顶部导航栏（固定）       │   │
│  └─────────────────────────────┘   │
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │   iframe容器                │   │
│  │   ┌──────────┐              │   │
│  │   │ MG系统   │ (隐藏/显示)  │   │
│  │   └──────────┘              │   │
│  │   ┌──────────┐              │   │
│  │   │ Lite系统 │ (隐藏/显示)  │   │
│  │   └──────────┘              │   │
│  │                             │   │
│  └─────────────────────────────┘   │
└─────────────────────────────────────┘
```

### 关键技术

1. **iframe嵌入**
   - 使用iframe加载两个系统
   - 通过CSS控制显示/隐藏
   - 保持系统状态不丢失

2. **LocalStorage共享**
   - 两个系统共享数据存储
   - 数据实时同步
   - 无需后端支持

3. **智能切换**
   - 首次加载时才请求iframe
   - 切换时直接显示/隐藏
   - 记忆上次访问的系统

4. **优雅降级**
   - 加载动画提示
   - 错误处理机制
   - 兼容性保证

## 📞 问题反馈

### 常见问题

**Q1: 切换系统后数据没有更新？**
A: 点击「刷新」按钮重新加载系统。

**Q2: 重置数据后看不到效果？**
A: 重置后需要点击「刷新」按钮才能看到效果。

**Q3: 可以同时打开两个系统吗？**
A: 统一入口一次只能显示一个系统，如需同时查看，请直接访问对应系统。

**Q4: 统一入口和直接访问有什么区别？**
A: 统一入口提供了更便捷的切换方式，但功能完全相同。

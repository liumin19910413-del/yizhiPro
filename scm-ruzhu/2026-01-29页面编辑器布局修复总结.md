# 2026-01-29 页面编辑器布局修复总结

## 问题描述

用户反馈在页面编辑器中遇到以下问题：
1. ❌ 无法一眼看到已用组件列表
2. ❌ 选中组件后无法在右侧进行配置
3. ❌ 右侧被产品说明文档遮挡

## 根本原因

产品说明文档面板默认展开（400px宽），完全遮挡了右侧的配置面板（320px宽），导致用户无法看到和使用配置功能。

## 解决方案

### 修改内容

1. **文档面板默认收起**
   - 修改前：默认展开，宽度400px
   - 修改后：默认收起，宽度40px，只显示一个竖向按钮

2. **文档面板样式优化**
   - 收起时：显示竖向文字"产品说明"和"▶"图标
   - 展开时：显示完整文档和"◀"图标
   - 点击按钮可以切换展开/收起状态

3. **配置面板完全可见**
   - 右侧配置面板（320px）不再被遮挡
   - 默认显示页面设置（8个配置项）
   - 选中组件后显示组件配置

### 代码修改

**文件：** `prototype/mg/app.js` 和 `dist/mg/app.js`

**修改1：文档面板HTML结构**
```javascript
// 修改前
<div class="editor-doc-panel" id="editor-doc-panel">

// 修改后
<div class="editor-doc-panel collapsed" id="editor-doc-panel">
```

**修改2：文档面板CSS样式**
```css
/* 收起状态 */
.editor-doc-panel.collapsed {
    width: 40px;
    border-left: none;
}

.editor-doc-panel.collapsed .doc-panel-content {
    display: none;
}

.editor-doc-panel.collapsed .doc-panel-header {
    writing-mode: vertical-rl;
    text-align: center;
}
```

**修改3：按钮布局**
```html
<!-- 修改前 -->
<div class="doc-panel-header">
    <span>产品说明</span>
    <button onclick="toggleEditorDocPanel()">
        <span id="doc-toggle-icon">◀</span>
    </button>
</div>

<!-- 修改后 -->
<div class="doc-panel-header">
    <button onclick="toggleEditorDocPanel()">
        <span id="doc-toggle-icon">▶</span>
        <span>产品说明</span>
    </button>
</div>
```

## 技术细节

### CSS关键样式
```css
/* 文档面板默认收起 */
.editor-doc-panel.collapsed {
    width: 40px;
    border-left: none;

/* 收起时隐藏内容 */

    display: none;

    writing-mode: vertical-rl;
    text-align: center;
}
```

### JavaScript关键函数
```javascript
// 切换文档面板
function toggleEditorDocPanel() {
    const panel = document.getElementById('editor-doc-panel');
    const icon = document.getElementById('doc-toggle-icon');
    
    if (panel.classList.contains('collapsed')) {
        panel.classList.remove('collapsed');
        icon.textContent = '◀';
    } else {
        panel.classList.add('collapsed');
        icon.textContent = '▶';
    }
}
```

# 分账设置功能说明

## 功能概述

在Lite商户端的入驻管理中，为连锁品牌商户增加了"分账设置"功能，用于配置品牌总部的分账接收方和分成比例。

## 功能位置

**路径：** Lite商户后台 → 店商供应链 → 入驻管理 → 申请记录列表 → 操作列 → 分账设置

## 显示条件

只有满足以下条件的申请记录才会显示"分账设置"按钮：
- 申请记录中配置了品牌总部商户号（`brandMerchantId`）
- 品牌总部分成比例大于0（`brandRatio > 0`）

## 权限控制

分账设置功能有三种权限级别：

### 1. 不可见（none）
- 用户没有 `supply-chain-profit-setting` 的 `view` 权限
- 不显示"分账设置"按钮
- 完全无法访问此功能

### 2. 只能浏览（view）
- 用户有 `supply-chain-profit-setting` 的 `view` 权限
- 但没有 `edit` 权限
- 显示"分账设置"按钮
- 可以查看当前配置的分账接收方和分成比例
- 不能修改分账接收方和分成比例
- 弹窗中显示提示："您当前只有查看权限，如需修改请联系管理员"

### 3. 能修改（edit）
- 用户有 `supply-chain-profit-setting` 的 `edit` 权限
- 显示"分账设置"按钮
- 可以查看和修改分账接收方
- 可以修改品牌总部分成比例（0-100%）
- 可以保存修改

## 功能详情

### 分账设置弹窗内容

1. **商户信息**
   - 商户号
   - 商户名称
   - 品牌总部名称和商户号

2. **分账接收方设置**（可编辑模式）
   - 分账接收方下拉选择框
   - 显示品牌总部配置的所有激活状态的分账接收方
   - 接收方信息包括：名称、类型（商户号/银行账户）、账户信息
   - 品牌总部分成比例输入框（0-100%，支持小数）
   - 说明文字："设置品牌总部从商家收益中分走的比例，范围：0-100%"

3. **分账接收方设置**（只读模式）
   - 当前分账接收方（只读显示）
   - 品牌总部分成比例（只读显示）

4. **可用分账接收方列表**
   - 表格形式展示所有可用的分账接收方
   - 包括：接收方名称、类型、账户信息、状态

### 操作逻辑

- 分账设置是按**商户号**维度进行的
- 一个商户号只能配置一个分账接收方
- 可以同时修改分账接收方和分成比例
- 分成比例范围：0-100%，支持小数（如10.5%）
- 修改分账接收方和分成比例会更新申请记录中的 `profitReceiver` 和 `brandRatio` 字段
- 保存后会记录更新时间

## 数据结构

### 申请记录（applications）
```javascript
{
  id: 'APP202501001',
  merchantId: 'YZF202501001',
  brandMerchantId: 'YZF202501004',  // 品牌总部商户号
  brandRatio: 10,                    // 品牌总部分成比例（可修改）
  profitReceiver: 'PR001',           // 分账接收方ID（可修改）
  updateTime: '2025-01-20 10:00:00', // 更新时间
  // ... 其他字段
}
```

### 分账接收方（profitReceivers）
```javascript
{
  id: 'PR001',
  merchantId: 'YZF202501004',        // 所属商户号（品牌总部）
  receiverName: '美丽集团总部',
  receiverAccount: 'YZF202501004',
  receiverType: 'merchant',          // merchant | bank
  bankName: '中国工商银行',
  bankAccount: '6222021234567890123',
  accountHolder: '广州美丽集团有限公司',
  status: 'active',                  // active | inactive
  createTime: '2024-12-01 10:00:00',
  remark: '备注信息'
}
```

## 权限配置示例

### 店长角色（完整权限）
```javascript
{
  name: '店长',
  permissions: {
    'supply-chain-apply': ['view', 'create', 'edit'],
    'supply-chain-profit-setting': ['view', 'edit']  // 可查看和修改
  }
}
```

### 财务角色（只读权限）
```javascript
{
  name: '财务',
  permissions: {
    'supply-chain-apply': ['view'],
    'supply-chain-profit-setting': ['view']  // 只能查看
  }
}
```

### 收银员角色（不可见）
```javascript
{
  name: '收银员',
  permissions: {
    'supply-chain-orders': ['view', 'edit']
    // 没有 supply-chain-profit-setting 权限，不显示分账设置按钮
  }
}
```

## 测试场景

### 场景1：店长修改分账设置
1. 以店长角色登录
2. 进入"入驻管理"页面
3. 找到有品牌总部分成的申请记录
4. 点击"分账设置"按钮
5. 在下拉框中选择新的分账接收方
6. 修改品牌总部分成比例（如从10%改为12%）
7. 点击"保存"
8. 验证保存成功提示
9. 重新打开分账设置，验证已保存

### 场景2：财务查看分账设置
1. 以财务角色登录
2. 进入"入驻管理"页面
3. 找到有品牌总部分成的申请记录
4. 点击"分账设置"按钮
5. 查看当前配置的分账接收方和分成比例
6. 验证无法修改（没有输入框）
7. 看到提示："您当前只有查看权限，如需修改请联系管理员"

### 场景3：收银员不可见
1. 以收银员角色登录
2. 进入"入驻管理"页面（如果有权限）
3. 验证申请记录列表中没有"分账设置"按钮

### 场景4：单店商户（无品牌总部）
1. 查看单店商户的申请记录
2. 验证没有"分账设置"按钮（因为没有品牌总部分成）

## 其他变更

   - 申请详情弹窗：删除"申请编号"字段

2. **MG管理后台**
   - 商户入驻 → 入驻审批：表格中没有"申请编号"列（已确认）
   - 申请详情弹窗：没有显示"申请编号"（已确认）

## 注意事项

1. 分账接收方必须由品牌总部在MG管理后台预先配置
2. 只有状态为"激活"（active）的分账接收方才会显示在选择列表中
3. 如果品牌总部没有配置任何分账接收方，会显示提示信息
4. 分账设置的修改会立即生效，影响后续的订单分账
5. 权限验证在前端进行，实际生产环境需要后端API进行二次验证

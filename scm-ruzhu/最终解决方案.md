# 最终解决方案 - 系统切换功能

## 🎯 问题回顾

用户需求：在同一个浏览器窗口中切换MG管理后台和Lite商户后台，不需要打开多个标签页。

尝试过的方案：
1. ❌ iframe嵌入方式 - 出现白屏问题
2. ❌ 修改资源路径 - 仍然白屏
3. ❌ 创建iframe专用HTML - 仍然白屏
4. ❌ 动态加载内容 - 复杂且不可靠

## ✅ 最终方案

**使用页面跳转方式实现单窗口系统切换**

### 核心思路
- 统一入口页面展示两个系统的卡片
- 点击卡片在当前窗口跳转到对应系统
- 每个系统添加"返回系统选择"按钮
- 利用浏览器原生的页面导航功能

### 实现文件

#### 1. 统一入口：`index-switch.html`
```
功能：展示系统选择卡片
位置：prototype/index-switch.html
访问：http://localhost:8000/index-switch.html
```

#### 2. MG系统：`mg/index.html`（已修改）
```
修改：添加"返回系统选择"按钮
位置：侧边栏顶部
功能：点击返回统一入口
```

#### 3. Lite系统：`lite/index.html`（已修改）
```
修改：添加"返回系统选择"按钮
位置：侧边栏顶部
功能：点击返回统一入口
```

## 📋 使用方法

### 启动服务器
```bash
cd prototype
python3 -m http.server 8000
```

### 访问系统
```
1. 打开浏览器访问：http://localhost:8000/index-switch.html
2. 点击"MG管理后台"或"Lite商户后台"卡片
3. 在系统中操作
4. 点击侧边栏的"返回系统选择"按钮
5. 选择另一个系统继续操作
```

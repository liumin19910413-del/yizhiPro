- ✅ 支持终止正在生效的价格

#### 1.3 价格计算逻辑
```
假设商品售价1000元，手续费率0.6%：
- 手续费 = 1000 × 0.6% = 6元
- 可分配金额 = 1000 - 6 = 994元
- 商家收益 = 994 × 10% = 99.4元
- 品牌总部 = 99.4 × 10% = 9.94元
- 商家实际收益 = 99.4 - 9.94 = 89.46元
- 私域运营 = 994 × 20% = 198.8元
- 供应链平台 = 994 × 70% = 695.8元
```

## 业务规则

### 1. 商品组合规则
- 只能组合同一供应商的产品
- 至少选择一个产品
- 销售价格不能为空

### 2. 价格设置规则
- 三方分成比例之和必须等于100%
- 结束时间必须大于开始时间
- 品牌总部分成占商家收益的百分比

### 3. 商品状态规则
- 已上架商品不可删除
- 只能删除已下架的商品
- 删除会同时删除关联价格

### 4. 操作权限
- 所有操作记录操作人
- 操作日志不可删除
- 价格终止后不可恢复

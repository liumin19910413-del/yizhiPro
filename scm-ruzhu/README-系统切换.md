# 系统切换功能 - 完整说明

## 🎉 问题已解决！

用户需求：**在同一个浏览器窗口中切换MG和Lite系统，无需多个标签页**

解决方案：**使用页面跳转方式实现单窗口系统切换**

## 🚀 立即使用

### 访问地址
```
http://localhost:8000/index-switch.html
```

### 操作步骤
1. 点击卡片进入系统
2. 使用系统功能
3. 点击"返回系统选择"按钮
4. 选择另一个系统

## 🔧 技术实现

### 统一入口
```javascript
// 点击卡片跳转到系统
function goToSystem(system) {
    window.location.href = system + '/index.html';
}
```

### 返回按钮
```html
<!-- 在系统页面添加返回按钮 -->
<button onclick="window.location.href='../index-switch.html'">
    ← 返回系统选择
</button>
```

## 💡 常见问题

### Q: 为什么不用iframe？
A: iframe会导致白屏问题，经过多次尝试无法解决。

### Q: 切换会丢失数据吗？
A: 不会，数据存储在localStorage中。

### Q: 可以用浏览器后退吗？
A: 可以，完全支持浏览器前进/后退。

### Q: 切换速度快吗？
A: 很快，页面跳转几乎是即时的。

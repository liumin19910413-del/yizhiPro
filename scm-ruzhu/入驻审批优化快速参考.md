# MG端商户入驻审批优化 - 快速参考

## 🎯 核心功能

### 筛选条件（8个）
1. **申请时间** - 开始/结束日期
2. **审核时间** - 开始/结束日期
3. **伊智付分账状态** - 全部/已开通/未开通
4. **营业执照经营范围** - 全部/符合/不符合
5. **审核状态** - 全部/待审核/已通过/已拒绝
6. **签约状态** - 全部/已签约/待签约
7. **搜索** - 商户号/商户名称
8. **操作** - 查询/重置

### 列表字段（16个）
1. 商户号
2. 收款主体
3. 关联商户名称
4. 关联连锁名称
5. 申请账号
6. 伊智付分账状态
7. 营业执照经营范围
8. 经销单位
9. 经销员工
10. 审核状态
11. 签约状态
12. 商品分佣比例（查看）
13. 最新审核时间
14. 审核人
15. 日志（查看）
16. 操作

### 弹窗功能（3个）
1. **商品分佣比例** - 查看分成配置
2. **审核日志** - 查看操作历史
3. **审核拒绝** - 输入拒绝原因

### 操作按钮（4个）
1. **详情** - 查看完整信息
2. **审核通过** - 审核并设置分成
3. **审核拒绝** - 拒绝并说明原因
4. **提醒签约员工** - 发送签约提醒

## 📝 使用示例

### 筛选已开通分账且经营范围符合的记录
```javascript
1. 选择"伊智付分账状态" = "已开通"
2. 选择"营业执照经营范围" = "符合"
3. 点击"查询"
```

### 查看某个申请的分佣比例
```javascript
1. 找到目标记录
2. 点击"商品分佣比例"列的"查看"按钮
3. 在弹窗中查看详细信息
```

### 审核拒绝某个申请
```javascript
1. 找到待审核记录
2. 点击"审核拒绝"按钮
3. 输入拒绝原因
4. 点击"确认拒绝"
```

### 提醒商户签约
```javascript
1. 找到待签约记录
2. 点击"提醒签约员工"按钮
3. 系统发送提醒通知
```

## 💡 提示

### 筛选技巧
- 多条件组合使用AND逻辑
- 时间范围可以只设置开始或结束
- 搜索支持商户号和名称
- 重置按钮一键清空所有条件

### 数据查看
- 分佣比例：只有已审核通过的记录才有数据
- 审核日志：所有记录都有日志
- 操作按钮：根据状态动态显示

### 常见问题
- 筛选后无数据？检查条件是否过严
- 分佣比例无数据？该记录可能未审核
- 按钮不显示？检查记录状态

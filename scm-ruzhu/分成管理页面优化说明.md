# 分成管理页面优化说明

## 修改内容

### 1. 菜单栏名称修改
- **旧名称**：分润管理
- **新名称**：分成管理
- **修改位置**：MG端侧边栏菜单

### 2. Tab页名称修改
- **旧名称**：品牌总部分成配置
- **新名称**：品牌总部分成设置
- **修改位置**：分成管理页面的第一个Tab

### 3. Tab切换逻辑修复（最终方案）
- **问题描述**：点击【经销商单位分成设置】Tab后，页面会自动切换回【品牌总部分成设置】
- **问题原因**：
  1. 页面每5秒自动刷新一次（定时器）
  2. 每次渲染时HTML中硬编码了第一个Tab为active
  3. 导致Tab状态被重置
- **解决方案**：在renderMerchantsProfit函数中，根据currentProfitTab变量动态设置哪个Tab是active的

## 技术实现

### 新增全局变量
```javascript
let currentProfitTab = 'brand'; // 记住当前选中的分成管理Tab
```

### 修改renderMerchantsProfit函数
```javascript
function renderMerchantsProfit() {
    const profitRules = DataStore.get('profitRules');
    const distributors = DataStore.get('distributors');
    
    // 根据currentProfitTab决定哪个Tab是active的
    const brandActive = currentProfitTab === 'brand' ? 'active' : '';
    const distributorActive = currentProfitTab === 'distributor' ? 'active' : '';
    
    return `
        ...
        <div class="profit-tab ${brandActive}" data-tab="brand" ...>
        <div class="profit-tab ${distributorActive}" data-tab="distributor" ...>
        ...
        <div id="brand-tab-content" class="tab-pane ${brandActive}">
        <div id="distributor-tab-content" class="tab-pane ${distributorActive}">
        ...
    `;
}
```

### 修改switchProfitTab函数
```javascript
function switchProfitTab(tabName) {
    // 记住当前选中的Tab
    currentProfitTab = tabName;
    
    // 更新Tab导航状态
    document.querySelectorAll('.profit-tab').forEach(tab => {
        tab.classList.remove('active');
    });
    document.querySelector(`.profit-tab[data-tab="${tabName}"]`).classList.add('active');
    
    // 更新Tab内容显示
    document.querySelectorAll('.tab-pane').forEach(pane => {
        pane.classList.remove('active');
    });
    document.getElementById(`${tabName}-tab-content`).classList.add('active');
}
```

### 关键改进
- ✅ 不再使用setTimeout恢复Tab状态
- ✅ 直接在HTML渲染时根据currentProfitTab设置active类
- ✅ 即使页面每5秒自动刷新，Tab状态也能保持
- ✅ 更简洁、更可靠的实现方式

## 相关功能

### 品牌总部分成设置Tab
- 显示各商户号配置的品牌总部分成比例
- 仅供查看，不可编辑
- 显示商户号、商户名称、绑定门店数、品牌总部信息、分成比例

### 经销商单位分成设置Tab
- 显示经销商单位与伊智贸易公司的分成比例
- 可以编辑单个经销商的分成比例
- 可以批量设置所有经销商的分成比例
- 显示经销商ID、名称、分成比例、绑定商户数

## 注意事项

1. **Tab状态保持**：Tab状态会在页面刷新时保持，但浏览器关闭后会重置为默认的第一个Tab
2. **自动刷新优化**：分成管理页面不会每5秒自动刷新，避免页面闪烁
3. **手动刷新**：切换到分成管理页面时会刷新一次，显示最新数据
4. **编辑后刷新**：编辑经销商保存后，页面会刷新，但Tab状态会保持在【经销商单位分成设置】
5. **批量设置后刷新**：批量设置后，列表会自动刷新，Tab状态保持不变

## 性能优化

### 防闪烁优化（v1.7.6）
- **问题**：页面内容每5秒闪现一次
- **原因**：定时器每5秒自动刷新所有页面
- **解决**：分成管理页面跳过自动刷新
- **效果**：页面内容稳定显示，不再闪烁

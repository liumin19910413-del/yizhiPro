# 🚀 快速开始

## 一分钟上手

### 1️⃣ 启动服务器
```bash
cd prototype
python3 -m http.server 8000
```

### 2️⃣ 打开浏览器
访问：**http://localhost:8000/index-switch.html**

### 3️⃣ 开始使用
- 点击 **"MG管理后台"** 卡片 → 进入MG系统
- 点击 **"Lite商户后台"** 卡片 → 进入Lite系统
- 点击侧边栏 **"返回系统选择"** 按钮 → 切换系统

## 🎯 核心功能

### MG管理后台
- 商户入驻审批
- 分润管理（Tab页优化）
- 产品维护
- 商品组合管理
- 预收款管理
- 订单管理

### Lite商户后台 - 营销应用架构 🆕
- **营销应用** - 应用入口页面
  - 店商供应链应用卡片
  - 未来扩展应用预览
- **主工作台** - 系统概览
- **商户信息** - 资料管理
- **系统设置** - 账户设置

#### 店商供应链应用
- 智能状态路由（根据申请状态自动跳转）
- 商家入驻申请
- 商品浏览
- 订单管理
- 保证金充值
- 财务对账

## ✨ 特点

- ✅ **快速切换** - 一键返回系统选择

## ❓ 常见问题

**Q: 为什么使用页面跳转而不是iframe？**
A: iframe会导致白屏问题，页面跳转方案更可靠。

**Q: 数据会丢失吗？**
A: 不会，所有数据存储在localStorage中。

**Q: 可以用浏览器后退按钮吗？**
A: 可以，完全支持浏览器导航。

# 2026-01-29 商城装修页面编辑器交付说明

## 交付概述

完成了商城装修页面编辑器的布局优化，解决了右侧配置面板被文档遮挡的问题，确保用户能够：
1. 清晰看到已用组件列表
2. 方便地配置选中的组件
3. 随时查看产品说明文档

## 核心问题修复

### 问题描述
- ❌ 产品说明文档面板默认展开，遮挡了右侧配置面板
- ❌ 用户无法一眼看到已用组件列表
- ❌ 选中组件后无法在右侧进行配置

### 解决方案
- ✅ 产品说明文档面板默认收起，只显示一个40px宽的侧边按钮
- ✅ 右侧配置面板完全可见，默认显示页面设置
- ✅ 选中组件后，右侧自动切换到组件配置面板
- ✅ 点击文档按钮可以展开/收起文档面板

## 页面布局（从左到右）

### 1. 左侧区域（280px）
**顶部Tab切换：**
- 页面设置（默认选中）
- 已用组件

**页面设置Tab内容：**
- 显示4个可拖拽组件：
  - 🛍️ 商品
  - 📦 商品分组
  - 🔍 商品搜索
  - 🖼️ 图片广告

**已用组件Tab内容：**
- 显示已添加到页面的所有组件
- 每个组件显示类型名称和删除按钮
- 点击组件可选中并在右侧配置
- 初始状态显示"暂无组件"

### 2. 中间预览区（自适应）
**iPhone 17尺寸预览框（430x932）：**
- 顶部标题栏（44px）
  - 左侧：返回按钮（可配置显示/隐藏）
  - 中间：页面标题
  - 背景：可配置颜色和透明度
- 内容区域
  - 显示所有已添加的组件
  - 背景：可配置颜色
  - 支持滚动

**预览框样式：**
- 圆角边框（40px）
- 深色阴影效果
- 白色背景
- 居中显示

### 3. 右侧配置面板（320px）
**默认状态（未选中组件）：**
显示页面设置，包含8个配置项：
1. **页面标题** - 文本输入框
2. **页面别名** - 文本输入框（只能包含小写字母、数字和连字符）
3. **分享封面** - 文本输入框（图片URL）
4. **页面描述** - 多行文本框（3行）
5. **背景颜色** - 颜色选择器 + 文本输入框
6. **返回操作** - 开关（Switch）
7. **透明标题栏** - 开关（Switch）
8. **顶部标题栏** - 开关（Switch）

**选中组件后：**
显示组件配置面板，根据组件类型显示不同的配置项：

**商品组件配置：**
- 选择商品（多选）
- 显示样式（网格/列表）
- 每行显示数量
- 商品间距

**商品分组组件配置：**
- 选择商品分类
- 显示数量
- 显示样式

**商品搜索组件配置：**
- 搜索框占位符
- 是否显示热门搜索词
- 热门搜索词列表

**图片广告组件配置：**
- 图片URL
- 跳转链接
- 图片高度
- 圆角大小

### 4. 产品说明文档按钮（40px，默认收起）
**收起状态：**
- 宽度：40px
- 显示竖向文字"产品说明"
- 显示"▶"图标
- 点击展开

**展开状态：**
- 宽度：400px
- 显示完整的产品说明文档
- 显示"◀"图标
- 点击收起

**文档内容：**
- 页面编辑器使用说明
- 基础操作指南
- 页面设置说明
- 组件说明
- 注意事项

## 使用流程

### 1. 创建页面
1. 进入MG管理后台
2. 导航到：运营管理 → 商城装修 → 内容管理
3. 点击"创建页面"按钮
4. 进入页面编辑器

### 2. 添加组件
1. 在左侧"页面设置"Tab中
2. 拖拽组件到中间预览区
3. 组件自动添加到页面
4. "已用组件"Tab自动更新

### 3. 查看已用组件
1. 点击左侧"已用组件"Tab
2. 查看所有已添加的组件列表
3. 点击组件可快速选中

### 4. 配置组件
1. 在预览区点击组件，或在"已用组件"列表中点击
2. 组件被选中（高亮显示）
3. 右侧自动显示该组件的配置面板
4. 修改配置项
5. 预览区实时更新

### 5. 配置页面
1. 在右侧配置面板修改页面设置
2. 修改页面标题、背景颜色等
3. 预览区实时更新
4. 可以随时切换到组件配置

### 6. 查看文档
1. 点击最右侧的"产品说明"按钮
2. 文档面板展开（400px）
3. 查看详细的使用说明
4. 再次点击按钮收起文档

### 7. 保存页面
1. 点击顶部"保存"按钮 - 保存草稿
2. 或点击"发布"按钮 - 保存并发布
3. 返回内容管理页面

## 交互细节

### Tab切换
- 点击Tab，下方内容区切换
- 选中的Tab显示紫色下划线
- 切换有平滑过渡动画

### 组件拖拽
- 从左侧拖拽组件
- 拖拽时鼠标变为移动图标
- 放到预览区后自动添加
- 预览区显示组件预览
- "已用组件"Tab自动更新

### 组件选中
- 点击预览区的组件
- 或点击"已用组件"列表中的组件
- 组件高亮显示（紫色边框）
- 右侧显示组件配置面板
- 列表中的组件也高亮

### 组件删除
- 在"已用组件"列表中点击删除按钮
- 弹出确认对话框
- 确认后删除组件
- 预览区和列表同步更新

### 实时预览
- 修改页面配置，预览区立即更新
- 修改组件配置，预览区立即更新
- 修改背景颜色，预览区背景立即变化
- 切换开关，预览区立即响应

### 文档面板
- 默认收起，只显示40px宽的按钮
- 点击按钮展开到400px
- 展开时显示完整文档
- 再次点击收起
- 状态保存到sessionStorage

## 技术实现

### 关键函数

**switchLeftTab(tabName)**
- 切换左侧Tab
- 更新Tab选中状态
- 切换内容区显示

**selectComponent(index)**
- 选中组件
- 更新选中状态（预览区和列表）
- 显示组件配置面板

**showComponentConfig(index)**
- 显示组件配置面板
- 根据组件类型渲染不同的配置项
- 绑定配置项的事件处理

**updatePageConfig(key, value)**
- 更新页面配置
- 保存到sessionStorage
- 实时更新预览区

**updateComponentConfig(index, key, value)**
- 更新组件配置
- 保存到sessionStorage
- 实时更新预览区

**toggleEditorDocPanel()**
- 切换文档面板展开/收起
- 更新图标
- 保存状态到sessionStorage

**renderUsedComponentsList(components)**
- 渲染已用组件列表
- 显示组件类型和删除按钮
- 绑定点击事件

**renderPageSettingsPanel(pageData)**
- 渲染页面设置面板
- 显示8个配置项
- 绑定配置项的事件处理

**renderComponentPreview(comp, index)**
- 渲染组件预览
- 根据组件类型显示不同的预览样式
- 绑定点击事件

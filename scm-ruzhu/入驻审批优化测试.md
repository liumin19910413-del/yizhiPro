# MG端商户入驻审批优化 - 测试指南

## 功能概述

本次优化全面重构了MG端的商户入驻审批页面，增强筛选功能、优化列表展示、新增多个查看弹窗，大幅提升审核效率。

## 测试环境

- 访问地址：http://localhost:8000/index-switch.html
- 测试路径：MG端 → 商户入驻 → 入驻审批

## 数据验证

### LocalStorage数据检查

**打开浏览器控制台，执行以下命令**:

```javascript
// 查看所有申请记录
const apps = DataStore.get('applications');
console.log('申请记录:', apps);

// 查看商户信息
const merchants = DataStore.get('merchants');
console.log('商户信息:', merchants);

// 验证数据关联
apps.forEach(app => {
    const merchant = merchants.find(m => m.id === app.merchantId);
    console.log(`申请${app.id}:`, {
        merchantId: app.merchantId,
        merchantName: merchant?.name,
        hasSubAccount: merchant?.hasSubAccount,
        scope: merchant?.scope
    });
});
```

## 常见问题

### Q1：筛选后没有数据显示？

**A：** 检查：
1. 筛选条件是否过于严格
2. 测试数据是否符合筛选条件
3. 点击"重置"按钮恢复所有数据

### Q2：商品分佣比例显示"暂无数据"？

**A：** 这是正常的，待审核的记录还没有设置分佣比例。只有审核通过后才会有分佣比例数据。

### Q3：如何重置测试数据？

**A：** 在浏览器控制台执行：
```javascript
DataStore.reset();
location.reload();
```

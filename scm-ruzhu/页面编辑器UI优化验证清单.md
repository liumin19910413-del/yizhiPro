# 页面编辑器UI优化验证清单

## 前置步骤

### 1. 清除浏览器缓存（必须！）
请按照以下步骤清除缓存：

**Mac系统 - 强制刷新**：
- Chrome/Edge: `Command + Shift + R`
- Safari: `Command + Option + R`
- Firefox: `Command + Shift + R`

**或者使用开发者工具**：
1. 打开开发者工具：`F12` 或 `Command + Option + I`
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

### 2. 确认访问正确的URL
```
http://localhost:8000/dist/mg/index.html
```

### 3. 确认服务器正在运行
```bash
# 查看8000端口
lsof -i:8000

# 如果没有运行，启动服务器
cd /Users/chenjia/yz-chenjia/20260114-SCM
python3 -m http.server 8000
```

## 常见问题排查

### 问题1：看不到Tab切换
**原因**：浏览器缓存  
**解决**：强制刷新（`Command + Shift + R`）

### 问题2：预览区还是小尺寸
**原因**：浏览器缓存  
**解决**：清空缓存并硬性重新加载

### 问题3：配置项只有2个
**原因**：浏览器缓存  
**解决**：使用无痕模式访问

### 问题4：没有产品说明文档面板
**原因**：浏览器缓存  
**解决**：清除所有缓存

### 问题5：页面闪退
**原因**：已修复，如果还出现说明缓存未清除  
**解决**：强制刷新

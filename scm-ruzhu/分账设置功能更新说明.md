# 分账设置功能更新说明

## 📅 更新时间
2025-01-20

## 🔄 更新内容

### 1. 新增功能：品牌总部分成比例设置

**更新前：**
- 只能选择分账接收方
- 分成比例固定，不可修改
- 商户信息区显示"品牌总部分成比例10% (占商家收益)"

**更新后：**
- ✅ 可以选择分账接收方
- ✅ 可以修改品牌总部分成比例（0-100%）
- ✅ 支持小数输入（如10.5%）
- ✅ 删除了商户信息区的固定分成比例显示
- ✅ 分成比例输入框带有说明文字

### 2. 界面优化

#### 可编辑模式
```
分账接收方设置
├── 分账接收方 * [下拉选择框]
│   └── 说明：选择用于接收品牌总部分成的账户
└── 品牌总部分成比例 * [数字输入框] %（占商家收益）
    └── 说明：设置品牌总部从商家收益中分走的比例，范围：0-100%
```

#### 只读模式
```
当前分账接收方：美丽集团总部 - 商户号
品牌总部分成比例：10%（占商家收益）
```

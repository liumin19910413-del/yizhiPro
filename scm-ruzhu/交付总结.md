## 📋 使用说明

### 快速开始

```bash
# 方法1：直接打开
open prototype/mg/index.html

# 方法2：启动服务器
cd prototype
./start-server.sh
# 访问 http://localhost:8000/mg/index.html
```

### 核心操作

1. **创建商品组合**
   - 商品管理 → 商品组合 → + 创建
   - 选择供应商和产品
   - 设置价格并创建

2. **设置商品价格**
   - 商品详情 → 价格列表 → + 添加价格
   - 配置分成比例
   - 查看预览并保存

3. **设置地区限制**
   - 商品详情 → 不可发货地区
   - 选择地区或应用模板

4. **设置包邮规则**
   - 商品详情 → 包邮设置
   - 选择类型或应用模板

详细操作请参考：[快速参考.md](./快速参考.md)

# 分账设置功能交付说明

## 📦 交付内容

### 1. 新增功能
✅ **Lite商户端 - 分账设置功能**
- 位置：入驻管理 → 申请记录 → 操作列 → 分账设置
- 功能：为连锁品牌商户配置品牌总部的分账接收方
- 权限：支持三种权限级别（不可见、只读、可编辑）

### 2. 功能优化
✅ **删除申请编号字段**
- Lite商户端入驻管理列表
- Lite商户端申请详情弹窗
- MG管理后台（已确认无此字段）

## 🎯 功能特性

### 分账设置核心功能

#### 1. 显示逻辑
- 只对连锁品牌商户显示（有品牌总部且分成比例>0）
- 单店商户不显示此功能
- 按商户号维度进行配置

#### 2. 权限控制
| 权限 | 配置 | 效果 |
|------|------|------|
| 不可见 | 无 `supply-chain-profit-setting` view权限 | 不显示按钮 |
| 只读 | 有 view 权限，无 edit 权限 | 可查看，不可修改 |
| 可编辑 | 有 edit 权限 | 可查看和修改 |

#### 3. 功能界面
- **商户信息展示**：商户号、商户名称、品牌总部、分成比例
- **分账接收方设置**：下拉选择框（可编辑）或只读显示
- **可用接收方列表**：表格展示所有激活状态的接收方
- **操作按钮**：保存（可编辑）或关闭（只读）

#### 4. 数据管理
- 分账接收方由品牌总部在MG端预先配置
- 只显示状态为"激活"的接收方
- 支持商户号类型和银行账户类型
- 修改后立即生效

## 🔐 权限配置

### 在角色权限管理中配置

1. **进入权限配置**
   - Lite商户后台 → 门店配置 → 角色权限
   - 选择角色 → 配置权限

2. **找到分账设置权限节点**
   ```
   营销应用
   └── 店商供应链
       └── 入驻管理
           └── 分账设置
               ├── 查看
               └── 编辑
   ```

3. **权限组合**
   - 不可见：不勾选任何权限
   - 只读：只勾选"查看"
   - 可编辑：勾选"查看"和"编辑"

## ⚠️ 注意事项

### 使用限制
1. 分账接收方必须由品牌总部预先在MG端配置
2. 只有连锁品牌商户才能使用此功能
3. 一个商户号只能配置一个分账接收方
4. 修改后立即生效，影响后续订单分账

### 数据依赖
1. 申请记录必须有 `brandMerchantId` 和 `brandRatio > 0`
2. 品牌总部必须有激活状态的分账接收方
3. 分账接收方数据存储在 `profitReceivers` 表中

### 权限验证
1. 前端进行权限判断和UI控制
2. 生产环境需要后端API进行二次验证
3. 权限配置通过角色权限管理系统

## 📞 技术支持

### 问题排查
1. **分账设置按钮不显示**
   - 检查是否为连锁品牌商户
   - 检查 brandRatio 是否大于0
   - 检查用户权限配置

2. **无法保存分账设置**
   - 检查是否选择了分账接收方
   - 检查浏览器控制台错误信息
   - 检查 LocalStorage 是否可用

3. **权限不生效**
   - 检查角色权限配置
   - 检查用户角色分配
   - 刷新页面重新加载权限

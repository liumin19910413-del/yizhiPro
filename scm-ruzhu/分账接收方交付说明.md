## 🎯 功能特点

### 业务价值

1. **解决痛点**：商户申请时没有分账接收方账户无法完成申请
2. **简化流程**：在申请流程中直接添加，无需中断
3. **提高效率**：一次性提交所有信息，减少来回沟通
4. **提升体验**：智能提示，操作流畅

### 技术特点

1. **智能联动**：根据选择自动显示/隐藏相关字段
2. **数据关联**：分账接收方申请与入驻申请自动关联
3. **状态管理**：清晰的数据状态流转
4. **视觉反馈**：高亮显示、颜色区分、友好提示

## 📖 使用指南

### 商户端操作流程

1. 访问Lite端 → 营销应用 → 店商供应链
2. 点击"申请入驻"
3. 填写基本信息
4. 选择品牌总部商户号
5. 设置分成比例
6. 选择或添加分账接收方
7. 提交申请

### 平台端操作流程

1. 访问MG端 → 商户入驻 → 入驻审批
2. 找到待审核的申请
3. 点击"审核"按钮
4. 查看完整信息（包括分账接收方）
5. 填写审核意见
6. 通过或拒绝

## 🔍 注意事项

### 开发注意事项

1. **数据关联**：确保分账接收方申请的applicationId正确关联到入驻申请
2. **状态管理**：注意pending状态的处理
3. **表单验证**：必填项验证要完整
4. **错误处理**：提供友好的错误提示

### 使用注意事项

1. **必填项**：带红色星号的字段必须填写
2. **分成比例**：必须大于0才会显示分账接收方选择
3. **账户类型**：对公和对私账户的字段不同，注意区分
4. **数据保存**：使用LocalStorage，清除浏览器数据会丢失

## 📞 技术支持

### 常见问题

**Q：为什么看不到分账接收方选择？**
A：需要先选择品牌总部商户号并填写分成比例（>0）

**Q：添加分账接收方后为什么还是"暂无可用"？**
A：新添加的是"待审核"状态，会显示为"待审核的分账接收方"

**Q：MG端看不到分账接收方信息？**
A：检查入驻申请是否设置了品牌总部分成

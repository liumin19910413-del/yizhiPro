功能 Demo / PC 员工端

代客户下单完整演示流程

员工端先从订单页点击“代客户下单”，完成会员、商品、地址、价格配置后提交订单并复制支付链接。

1. 点击入口PC 右侧弹窗
2. 选择会员支持搜索或新建会员
3. 选择商品与改价支持数量、邮费、商品总价调整
4. 提交订单复制唯一付款链接
5. 客户点击链接支付支持会员卡切换与微信补差
功能 Demo / 移动员工端

移动端代下单流程

移动端员工通过商城订单页新增按钮条进入代客下单流程，底部固定提交按钮确保操作便捷。

1. 点击入口订单列表页新增按钮条
2. 选择会员搜索或新建客户
3. 商品配置搜索、加购、数量调整
4. 预览订单底部半屏确认单
5. 提交复制链接回到订单列表
功能 Demo / 客户支付端

客户支付流程

客户点击收到的付款链接后进入订单详情页，支持会员卡金抵用+微信支付组合付款方式。

1. 打开链接进入订单详情
2. 查看订单确认商品和地址
3. 会员卡判断足额 / 不足 / 无卡
4. 去付款0 元直付或微信补差
5. 支付成功切换待发货状态

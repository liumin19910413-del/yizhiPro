# Project Rules

This project is connected to the `yz-kb@personal` plugin, which provides access to the Yizhi knowledge base.

## Default Knowledge-Base Rule

For any user question about Yizhi Technology, Yizhi products, company positioning, product architecture, AI Operation Brain, Zhenwo, Siyubao, AI Store Manager, AI Sales Champion, AI digital employees, AI Dianshang, sales scripts, marketing copy, customer-facing introductions, PRD wording, or internal product口径, always query the `yz-kb` knowledge base first.

Do not answer these questions from model memory alone.

## How To Answer

- First search the Yizhi knowledge base through `yz-kb`.
- Base the answer on the retrieved knowledge-base content.
- If the knowledge base has no clear answer, say that the knowledge base did not provide a clear口径 before giving a best-effort suggestion.
- Prefer practical wording that a boss, salesperson, product manager, or market team can directly reuse.
- When refining customer-facing copy, keep the business essence sharp: avoid merely listing products; explain the operating logic, customer pain, and why Yizhi is different.

## Useful User Intent Shortcut

When the user says anything like "先查知识库", "查伊智知识库", "按知识库口径", "对外介绍", or asks "这个怎么讲", treat it as an instruction to use `yz-kb` before answering.

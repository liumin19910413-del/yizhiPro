# 分账设置功能 - 快速参考

## 🎯 功能位置
**Lite商户后台 → 店商供应链 → 入驻管理 → 申请记录 → 分账设置**

## 📋 显示条件
- ✅ 连锁品牌商户（有品牌总部）
- ✅ 品牌总部分成比例 > 0
- ❌ 单店商户不显示

## 🔐 三种权限

| 权限级别 | 权限配置 | 按钮显示 | 功能 |
|---------|---------|---------|------|
| **不可见** | 无 `view` 权限 | ❌ 不显示 | 完全无法访问 |
| **只读** | 有 `view` 权限 | ✅ 显示 | 只能查看，不能修改 |
| **可编辑** | 有 `edit` 权限 | ✅ 显示 | 可以查看和修改 |

## 🎨 界面说明

### 弹窗内容
1. **商户信息区**
   - 商户号
   - 商户名称
   - 品牌总部

2. **分账接收方设置**
   - 分账接收方下拉选择框（可编辑模式）
   - 品牌总部分成比例输入框（可编辑模式）
   - 当前配置显示（只读模式）

3. **可用接收方列表**
   - 表格展示所有可用接收方

## 💾 数据结构

```javascript
// 申请记录可修改字段
{
  profitReceiver: 'PR001',  // 分账接收方ID
  brandRatio: 10.5,         // 品牌总部分成比例（可修改）
  updateTime: '2025-01-20 10:00:00'
}
```

## 🔧 权限配置示例

```javascript
// 店长 - 完整权限
{
  'supply-chain-profit-setting': ['view', 'edit']
}

// 财务 - 只读权限
{
  'supply-chain-profit-setting': ['view']
}

// 收银员 - 不可见
{
  // 没有 supply-chain-profit-setting 权限
}
```

## 🚀 快速测试

1. **访问地址**
   ```
   http://localhost:8000/prototype/lite/index.html
   ```

2. **测试步骤**
   - 进入"店商供应链" → "入驻管理"
   - 找到有品牌总部的申请记录
   - 点击"分账设置"按钮
   - 选择分账接收方
   - 修改分成比例（如10.5%）
   - 保存

3. **验证结果**
   - 显示成功提示
   - 重新打开验证已保存
   - 检查 LocalStorage 数据

## ⚠️ 注意事项

1. 分账接收方必须由品牌总部预先配置
2. 只显示状态为"激活"的接收方
3. 按商户号维度配置，一个商户号一个接收方
4. 可以同时修改分账接收方和分成比例
5. 分成比例范围：0-100%，支持小数（如10.5%）
6. 修改立即生效，影响后续订单分账

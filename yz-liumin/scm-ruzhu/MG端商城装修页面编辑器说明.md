# MG端商城装修页面编辑器功能说明

## 功能概述

商城装修页面编辑器是MG管理后台的核心功能之一，用于可视化配置商城小程序的页面内容。通过拖拽组件的方式，快速搭建商城页面，无需编写代码。

## 访问路径

MG管理后台 → 运营管理 → 商城装修 → 内容管理 → 创建页面/编辑

## 编辑器界面布局

### 1. 顶部工具栏
- **返回按钮**：关闭编辑器，返回页面列表（未保存的更改会丢失，需确认）
- **页面名称**：显示当前编辑的页面名称
- **保存草稿**：保存页面但不发布，状态为draft
- **保存并发布**：保存页面并发布，状态为published

### 2. 左侧组件库
提供4种基础组件，可拖拽到中间预览区：
- 🛍️ **商品**：展示商城商品列表
- 📦 **商品分组**：按分类展示商品
- 🔍 **商品搜索**：提供商品搜索功能
- 🖼️ **图片广告**：展示广告图片

### 3. 中间预览区
- 手机框架预览，模拟真实小程序效果
- 显示页面组件的实时预览
- 点击组件可选中并在右侧配置
- 空白状态提示"从左侧拖拽组件到这里"

### 4. 右侧配置面板
- **页面设置**：配置页面名称和别名
- **组件设置**：配置选中组件的属性
- **删除组件**：删除选中的组件

## 组件详细说明

### 1. 商品组件

**功能说明**：展示商城商品列表，关联"商品管理-商城商品"Tab的商品数据

**配置项**：
- **显示样式**
  - 网格布局：2列显示，适合展示多个商品
  - 列表布局：单列显示，适合详细展示商品信息
  
- **选择商品**
  - 从商城商品列表中多选商品
  - 支持复选框批量选择
  - 实时显示已选择商品数量
  - 预览区实时更新显示效果

**预览效果**：
- 网格布局：显示商品图片、名称、价格，2列排列
- 列表布局：显示商品图片、名称、价格，单列排列
- 超过显示数量时提示"还有X个商品..."

**数据来源**：mallGoods表（商城商品数据）

### 2. 商品分组组件

**功能说明**：按分类展示商品，自动显示该分类下的商品

**配置项**：
- **选择分类**
  - 从商品分类列表中选择一个分类
  - 下拉选择框，单选
  
- **显示数量**
  - 设置该分类下显示的商品数量
  - 数字输入框，范围1-50
  - 默认值：10

**预览效果**：
- 显示分类名称
- 提示"显示该分类下前X个商品"
- 未选择分类时提示"请在右侧配置中选择商品分类"

**数据来源**：categories表（商品分类数据）

### 3. 商品搜索组件

**功能说明**：提供商品搜索功能，用户可搜索商品

**配置项**：
- **搜索框提示文字**
  - 自定义搜索框的placeholder文字
  - 文本输入框
  - 默认值："搜索商品"
  
- **显示热门搜索词**
  - 开关控制是否显示热门搜索词
  - 复选框
  - 默认值：开启

**预览效果**：
- 显示搜索框样式（圆角、灰色背景）
- 显示搜索图标和提示文字
- 开启热门搜索词时显示示例词（口红、面膜、精华）

### 4. 图片广告组件

**功能说明**：展示广告图片，支持跳转链接

**配置项**：
- **广告图片**
  - 输入图片URL
  - 文本输入框
  - 显示图片预览
  - 可删除已设置的图片
  - 提示：实际使用时需要上传图片
  
- **链接类型**
  - 无链接：仅展示图片，不可点击
  - 跳转商品：点击跳转到指定商品详情页
  - 跳转页面：点击跳转到商城内其他页面
  - 外部链接：点击跳转到外部URL
  
- **链接地址**
  - 根据链接类型填写对应的地址
  - 文本输入框
  - 仅在链接类型不为"无链接"时显示

**预览效果**：
- 有图片时显示图片预览
- 有链接时显示链接类型标签（🔗 商品/页面/链接）
- 无图片时显示占位符（🖼️ 请在右侧配置中设置广告图片）

## 操作流程

### 创建新页面
1. 在内容管理Tab点击"创建页面"按钮
2. 自动进入页面编辑器，创建一个未命名页面
3. 从左侧拖拽组件到中间预览区
4. 点击组件进行配置
5. 在右侧配置面板设置页面名称和别名
6. 点击"保存草稿"或"保存并发布"

### 编辑现有页面
1. 在内容管理Tab点击页面的"编辑"按钮
2. 进入页面编辑器，加载现有页面数据
3. 可以添加、删除、配置组件
4. 修改页面名称和别名
5. 点击"保存草稿"或"保存并发布"

### 添加组件
1. 从左侧组件库拖拽组件到中间预览区
2. 组件自动添加到页面底部
3. 显示成功提示"组件已添加"
4. 预览区实时显示组件

### 配置组件
1. 点击预览区的组件
2. 组件高亮显示（紫色边框、浅紫色背景）
3. 右侧配置面板切换到组件设置
4. 根据组件类型显示不同的配置项
5. 修改配置后预览区实时更新

### 删除组件
1. 选中要删除的组件
2. 在右侧配置面板点击"删除组件"按钮
3. 确认删除操作
4. 组件从页面中移除
5. 配置面板恢复到页面设置

### 保存页面
1. 点击"保存草稿"或"保存并发布"
2. 验证页面名称和别名（必填）
3. 验证别名格式（小写字母、数字、连字符）
4. 验证别名唯一性
5. 保存成功后返回页面列表

## 数据结构

### 页面数据（mallPages表）
```javascript
{
  id: 'PAGE' + 时间戳,           // 页面ID
  name: '页面名称',               // 页面名称
  alias: 'page-alias',           // 页面别名（URL路径）
  isHomePage: false,             // 是否为主页
  createTime: '2026-01-29 ...',  // 创建时间
  updateTime: '2026-01-29 ...',  // 更新时间
  lastEditor: '系统管理员',       // 最后修改人
  status: 'draft',               // 状态：draft草稿 | published已发布
  components: [                  // 页面组件配置数组
    {
      type: 'goods',             // 组件类型
      id: 'COMP' + 时间戳,       // 组件ID
      config: {                  // 组件配置
        displayStyle: 'grid',    // 显示样式
        selectedGoods: ['MALL001'] // 选中的商品ID数组
      }
    }
  ]
}
```

### 组件配置结构

**商品组件**：
```javascript
{
  type: 'goods',
  id: 'COMP...',
  config: {
    displayStyle: 'grid',        // 显示样式：grid网格 | list列表
    selectedGoods: ['MALL001']   // 选中的商品ID数组
  }
}
```

**商品分组组件**：
```javascript
{
  type: 'goods-group',
  id: 'COMP...',
  config: {
    selectedCategory: 'CAT001',  // 选中的分类ID
    displayCount: 10             // 显示数量
  }
}
```

**商品搜索组件**：
```javascript
{
  type: 'goods-search',
  id: 'COMP...',
  config: {
    placeholder: '搜索商品',     // 搜索框提示文字
    showHotWords: true           // 是否显示热门搜索词
  }
}
```

**图片广告组件**：
```javascript
{
  type: 'image-ad',
  id: 'COMP...',
  config: {
    imageUrl: 'https://...',     // 图片URL
    linkType: 'goods',           // 链接类型：none无 | goods商品 | page页面 | url外部
    linkUrl: 'MALL001'           // 链接地址
  }
}
```

## 业务规则

### 页面规则
1. 页面名称必填，不能为空
2. 页面别名必填，不能为空
3. 页面别名只能包含小写字母、数字和连字符
4. 页面别名必须唯一（排除自己）
5. 系统只能有一个主页
6. 主页不能删除

### 组件规则
1. 组件可以任意添加、删除、排序
2. 组件配置实时保存到sessionStorage
3. 组件配置修改后预览区实时更新
4. 删除组件需要确认操作

### 保存规则
1. 保存草稿：status设为draft，不影响线上页面
2. 保存并发布：status设为published，立即生效
3. 保存时更新updateTime和lastEditor
4. 新建页面保存后添加到mallPages表
5. 编辑页面保存后更新mallPages表

## 关联数据

### 商城商品（mallGoods表）
- 商品组件从此表获取商品数据
- 显示商品名称、价格、封面图
- 支持多规格商品

### 商品分类（categories表）
- 商品分组组件从此表获取分类数据
- 显示分类名称
- 用于筛选商品

## 技术实现

### 拖拽功能
- 使用HTML5 Drag and Drop API
- 组件设置draggable="true"
- 监听dragstart、dragover、drop事件
- 通过dataTransfer传递组件类型

### 实时预览
- 组件配置保存到sessionStorage
- 配置修改后重新渲染预览区
- 使用innerHTML更新DOM
- 保持选中状态

### 数据持久化
- 编辑过程中数据保存到sessionStorage
- 点击保存时写入localStorage（DataStore）
- 关闭编辑器时清除sessionStorage
- 支持草稿和发布两种状态

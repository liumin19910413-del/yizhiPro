# 分账设置新增接收方功能说明

## 功能概述

在Lite商户端的"入驻管理-申请记录"中，点击"分账设置"后，在"可用分账接收方列表"旁边新增了"新增分账接收方"按钮，支持直接添加对公或对私的分账接收方。

## 功能位置

**路径**: Lite商户端 → 营销应用 → 店商供应链 → 入驻管理 → 申请记录 → 分账设置 → 新增分账接收方

## 功能详情

### 1. 入口

在"分账设置"弹窗中，"可用分账接收方列表"标题右侧有一个"+ 新增分账接收方"按钮。

### 2. 弹窗内容

点击按钮后，弹出"新增分账接收方"弹窗，包含两种账户类型：

#### 对公账户

**必填字段**:
- 分账接收方名称
- 联系手机号
- 营业执照编号
- 营业执照名称
- 营业执照照片（图片，限制2M）
- 法人姓名
- 法人身份证号
- 法人身份证照片（人像面 + 国徽面，各限制2M）
- 收款账户证件照片（人像面 + 国徽面，各限制2M）

**选填字段**:
- 收款账户名称
- 收款账户行号
- 收款账户卡号
- 银行卡照片（图片，限制2M）
- 收款账户证件号

#### 对私账户

**必填字段**:
- 分账接收方名称
- 联系手机号
- 收款账户卡号
- 收款账户证件号
- 收款账户证件照片（人像面 + 国徽面，各限制2M）

**选填字段**:
- 收款账户姓名

### 3. 表单验证

- **手机号验证**: 必须是11位有效手机号（1开头）
- **身份证号验证**: 必须是18位有效身份证号
- **文件大小验证**: 所有图片文件不能超过2M
- **文件类型验证**: 只能上传图片文件
- **必填项验证**: 所有标记为必填的字段都必须填写

### 4. 文件上传

- 点击上传区域可选择文件
- 上传后显示图片预览
- 可以点击预览图右上角的"×"删除已上传的文件
- 文件大小限制：2M
- 支持的格式：所有图片格式（jpg, png, gif等）

### 5. 提交流程

1. 选择账户类型（对公/对私）
2. 填写所有必填字段
3. 上传必需的图片文件
4. 点击"提交"按钮
5. 系统验证数据
6. 保存成功后显示提示信息
7. 自动刷新页面，新增的接收方会出现在"可用分账接收方列表"中

## 技术实现

### 新增函数

1. **showAddProfitReceiverModal(brandMerchantId)**
   - 显示新增分账接收方弹窗
   - 参数：品牌商户ID

2. **toggleReceiverType(type)**
   - 切换对公/对私表单显示
   - 参数：'company' 或 'personal'

3. **handleFileUpload(input, previewId)**
   - 处理文件上传和预览
   - 验证文件大小和类型
   - 显示图片预览

4. **removeUploadedFile(inputId, previewId)**
   - 移除已上传的文件
   - 清空input和预览区域

5. **submitNewProfitReceiver(brandMerchantId)**
   - 提交新增分账接收方表单
   - 验证所有必填项
   - 保存到DataStore
   - 刷新页面

## 注意事项

1. **文件上传**
   - 当前为前端模拟，实际项目中需要对接真实的文件上传接口
   - 文件数据存储在DataStore中，刷新页面后会丢失

2. **数据持久化**
   - 当前使用localStorage存储，清除浏览器数据会丢失
   - 实际项目中需要对接后端API

3. **权限控制**
   - 当前所有用户都可以新增接收方
   - 实际项目中可能需要根据用户权限控制

4. **表单重置**
   - 关闭弹窗后，表单数据会清空
   - 如需保留草稿，需要额外实现

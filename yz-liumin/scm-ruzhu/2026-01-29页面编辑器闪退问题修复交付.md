# 2026-01-29 页面编辑器闪退问题修复交付

## 问题回顾

**用户反馈**：
> 目前出现的问题，点击"创建页面"这个按钮进去后，过了一会儿会自动闪退回来一级菜单栏页面

**问题影响**：
- 无法正常使用页面编辑器
- 无法添加和配置组件
- 无法保存页面
- 严重影响用户体验

## 修复内容

### 1. 问题定位
通过代码分析发现，系统中有一个定时器每5秒自动刷新页面：
```javascript
setInterval(() => {
    const excludePages = ['merchants', 'merchants-profit', 'goods', 'products', 'mall-goods'];
    if (!excludePages.includes(currentPage)) {
        renderPage(currentPage);
    }
}, 5000);
```

当用户进入页面编辑器时，`currentPage`仍然是`'mall-decoration'`，不在排除列表中，导致5秒后自动刷新，编辑器被关闭。

### 2. 修复方案
添加全局标记`isInPageEditor`来标识是否在页面编辑器中：

**步骤1：添加全局标记**
```javascript
let isInPageEditor = false; // 标记是否在页面编辑器中
```

**步骤2：修改定时器逻辑**
```javascript
// 在定时器中检查标记
if (!excludePages.includes(currentPage) && !isInPageEditor) {
    renderPage(currentPage);
}
```

**步骤3：进入编辑器时设置标记**
```javascript
function showPageEditor(pageId) {
    // 设置标记，防止自动刷新
    isInPageEditor = true;
    // ...
}
```

**步骤4：退出编辑器时清除标记**
```javascript
function closePageEditor() {
    // 清除标记，恢复自动刷新
    isInPageEditor = false;
    // ...
}

function savePage(status) {
    // 清除标记，恢复自动刷新
    isInPageEditor = false;
    // ...
}
```

## 技术说明

### 为什么会闪退
1. 系统有定时器每5秒刷新页面
2. 编辑器是全屏覆盖的独立界面
3. 刷新会导致编辑器被销毁
4. 用户看到的效果就是"闪退"

### 为什么这样修复
1. 使用标记而不是修改currentPage
   - 保持currentPage的语义正确
   - 不影响其他功能
   
2. 在进入和退出时设置标记
   - 确保编辑器期间不刷新
   - 退出后立即恢复刷新
   
3. 同时修改定时器和storage监听
   - 全面防止自动刷新
   - 保证修复的完整性

### 优点
- ✅ 简单有效
- ✅ 改动最小
- ✅ 不影响其他功能
- ✅ 易于维护
- ✅ 可扩展（其他编辑器可复用）

## 代码对比

### 修改前
```javascript
// 没有标记
let currentPage = 'dashboard';
let currentProfitTab = 'brand';
let currentMerchantsTab = 'audit';

// 定时器会刷新所有非排除页面
setInterval(() => {
    const excludePages = ['merchants', 'merchants-profit', 'goods', 'products', 'mall-goods'];
    if (!excludePages.includes(currentPage)) {
        renderPage(currentPage);  // 会导致编辑器闪退
    }
}, 5000);

// 进入编辑器时没有设置标记
function showPageEditor(pageId) {
    const content = document.getElementById('page-content');
    content.innerHTML = renderPageEditor(pageData, isNewPage);
    initPageEditor(pageData);
}
```

### 修改后
```javascript
// 添加标记
let currentPage = 'dashboard';
let currentProfitTab = 'brand';
let currentMerchantsTab = 'audit';
let isInPageEditor = false; // 新增标记

// 定时器检查标记
setInterval(() => {
    const excludePages = ['merchants', 'merchants-profit', 'goods', 'products', 'mall-goods'];
    if (!excludePages.includes(currentPage) && !isInPageEditor) {  // 检查标记
        renderPage(currentPage);
    }
}, 5000);

// 进入编辑器时设置标记
function showPageEditor(pageId) {
    isInPageEditor = true;  // 设置标记，防止刷新
    const content = document.getElementById('page-content');
    content.innerHTML = renderPageEditor(pageData, isNewPage);
    initPageEditor(pageData);
}

// 退出编辑器时清除标记
function closePageEditor() {
    if (confirm('确定要关闭编辑器吗？未保存的更改将丢失。')) {
        sessionStorage.removeItem('editingPage');
        sessionStorage.removeItem('isNewPage');
        isInPageEditor = false;  // 清除标记，恢复刷新
        renderPage('mall-decoration');
    }
}

function savePage(status) {
    // ... 保存逻辑 ...
    isInPageEditor = false;  // 清除标记，恢复刷新
    renderPage('mall-decoration');
}
```

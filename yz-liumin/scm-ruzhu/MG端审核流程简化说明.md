# MG端审核流程简化说明

## 更新日期
2026年1月22日

## 更新概述
在MG管理后台的"商户入驻-入驻审批"模块中，简化了审核申请流程，去掉了"品牌总部分成比例"字段。

## 修改位置

**路径**: MG管理后台 → 商户入驻 → 入驻审批 → 点击"审核"按钮 → 审核申请弹窗

### 2. 审核通过函数（approveApp函数）

**修改前：**
```javascript
function approveApp(appId) {
    const brandRatioInput = document.getElementById('brandRatio');
    const brandRatio = brandRatioInput ? parseFloat(brandRatioInput.value) : 0;
    
    // ...
    
    if (app.brandMerchantId && brandRatio > 0) {
        contracts.push({ 
            type: 'brand', 
            name: '店商供应链合作协议-品牌总部', 
            signed: false, 
            signTime: null 
        });
    }
    
    DataStore.update('applications', appId, {
        // ...
        brandRatio,
        // ...
    });
}
```

**修改后：**
```javascript
function approveApp(appId) {
    // 不再读取brandRatio输入
    
    // ...
    
    // 不再根据brandRatio生成品牌总部协议
    
    DataStore.update('applications', appId, {
        // ...
        brandRatio: 0,  // 默认为0
        // ...
    });
}
```

## 数据变更

### applications表字段变化

| 字段名 | 修改前 | 修改后 | 说明 |
|--------|--------|--------|------|
| brandRatio | 从输入框读取（0-100） | 固定为0 | 品牌总部分成比例 |
| contracts | 可能包含品牌总部协议 | 只包含供应链和私域运营协议 | 协议列表 |

### 协议列表变化

**修改前：**


## 业务影响

### 1. 审核流程

- ✅ 审核人员无需再设置品牌总部分成比例
- ✅ 简化了审核操作步骤
- ✅ 减少了审核时的决策点

### 2. 协议签署

- ✅ Lite端商户无需签署品牌总部协议
- ✅ 只需签署供应链平台和私域运营平台两份协议

### 3. 分成计算

- ⚠️ 品牌总部分成比例默认为0
- ⚠️ 如需设置品牌总部分成，需在后续单独配置
- ⚠️ 与Lite端的简化流程保持一致

## 兼容性说明

### 历史数据

- 已审核通过的申请记录中的 `brandRatio` 字段保持不变
- 新审核的申请记录 `brandRatio` 默认为0

### 后续配置

- 品牌总部分成可在"分成管理"模块中单独配置
- 支持在申请通过后修改分成比例

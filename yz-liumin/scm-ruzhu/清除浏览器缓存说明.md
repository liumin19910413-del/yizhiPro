# 清除浏览器缓存说明

## 问题
刷新页面后代码没有更新，仍然显示旧的界面。

## 原因
浏览器缓存了旧的JavaScript文件（`lite/app.js`），需要强制刷新清除缓存。

## 解决方法

### 方法1：强制刷新（推荐）

#### Chrome / Edge / Firefox (Windows/Linux)
按住 `Ctrl + Shift + R` 或 `Ctrl + F5`

#### Chrome / Edge / Firefox (Mac)
按住 `Cmd + Shift + R`

#### Safari (Mac)
按住 `Cmd + Option + R`

### 方法2：清空缓存并硬性重新加载

#### Chrome / Edge
1. 打开开发者工具（F12 或 右键 → 检查）
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

#### Firefox
1. 打开开发者工具（F12）
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

#### Safari
1. 打开开发者工具（Cmd + Option + I）
2. 点击"网络"标签
3. 勾选"停用缓存"
4. 刷新页面

### 方法3：清除所有缓存

#### Chrome / Edge
1. 按 `Ctrl + Shift + Delete` (Windows) 或 `Cmd + Shift + Delete` (Mac)
2. 选择"缓存的图片和文件"
3. 时间范围选择"全部时间"
4. 点击"清除数据"

#### Firefox
1. 按 `Ctrl + Shift + Delete` (Windows) 或 `Cmd + Shift + Delete` (Mac)
2. 选择"缓存"
3. 时间范围选择"全部"
4. 点击"立即清除"

#### Safari
1. 菜单栏 → Safari → 偏好设置
2. 点击"高级"标签
3. 勾选"在菜单栏中显示开发菜单"
4. 菜单栏 → 开发 → 清空缓存
5. 或者按 `Cmd + Option + E`

### 方法4：禁用缓存（开发时使用）

#### Chrome / Edge / Firefox
1. 打开开发者工具（F12）
2. 打开设置（F1 或 点击齿轮图标）
3. 勾选"Disable cache (while DevTools is open)"
4. 保持开发者工具打开状态
5. 刷新页面

#### Safari
1. 打开开发者工具（Cmd + Option + I）
2. 点击"网络"标签
3. 勾选"停用缓存"
4. 保持开发者工具打开状态
5. 刷新页面

## 验证是否生效

### 方法1：检查控制台
1. 打开开发者工具（F12）
2. 切换到"控制台"标签
3. 输入以下代码并回车：
```javascript
console.log(typeof showAddEditReceiverModal);
```
4. 如果显示 `function`，说明新代码已加载
5. 如果显示 `undefined`，说明还是旧代码，需要继续清除缓存

### 方法2：检查文件加载时间
1. 打开开发者工具（F12）
2. 切换到"网络"标签
3. 刷新页面
4. 找到 `app.js` 文件
5. 查看"大小"列，如果显示"(from disk cache)"或"(from memory cache)"，说明使用了缓存
6. 如果显示实际文件大小（如"150 KB"），说明重新加载了文件

### 方法3：检查函数内容
1. 打开开发者工具（F12）
2. 切换到"控制台"标签
3. 输入以下代码并回车：
```javascript
console.log(showAddReceiverModal.toString());
```
4. 查看输出，如果包含 `showAddEditReceiverModal()`，说明新代码已加载
5. 如果包含大量HTML代码，说明还是旧代码

## 常见问题

### Q1：强制刷新后还是没变化
**解决**：
1. 关闭所有浏览器窗口
2. 重新打开浏览器
3. 访问页面

### Q2：清除缓存后页面报错
**解决**：
1. 再次强制刷新（Ctrl + Shift + R）
2. 检查控制台是否有JavaScript错误
3. 如果有错误，截图发给开发人员

### Q3：开发者工具中看不到"清空缓存并硬性重新加载"选项
**解决**：
1. 确保开发者工具已打开（F12）
2. 右键点击地址栏左侧的刷新按钮（不是工具栏的刷新按钮）
3. 应该会出现下拉菜单

### Q4：Safari中找不到"停用缓存"选项
**解决**：
1. 确保已启用开发菜单（Safari → 偏好设置 → 高级 → 勾选"在菜单栏中显示开发菜单"）
2. 打开开发者工具（Cmd + Option + I）
3. 点击"网络"标签
4. 在顶部工具栏中找到"停用缓存"复选框

## 开发建议

在开发期间，建议：
1. 始终保持开发者工具打开
2. 启用"禁用缓存"选项
3. 每次修改代码后使用强制刷新
4. 定期清空浏览器缓存

## 服务器缓存

如果使用了HTTP服务器（如Python的http.server），可能还需要：

### 重启服务器
```bash
# 停止服务器（Ctrl + C）
# 重新启动
cd prototype
python3 -m http.server 8000
```

### 或者使用无缓存启动
```bash
# 使用Python启动时添加无缓存头
python3 -m http.server 8000 --bind 127.0.0.1
```

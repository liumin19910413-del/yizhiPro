# 2026-01-29 页面编辑器UI优化说明

## 优化概述

根据用户需求和参考截图，对页面编辑器进行了全面的UI优化，提升了用户体验和视觉效果。

## 优化内容

### 一、产品说明文档位置调整

**优化前**：
- 产品说明文档显示在内容管理页面的右侧

**优化后**：
- ✅ 产品说明文档移到页面编辑器内部
- ✅ 位于编辑器右侧，可折叠显示
- ✅ 点击按钮可展开/收起文档面板
- ✅ 状态会保存到sessionStorage

**优点**：
- 在编辑器内即可查看文档，无需切换页面
- 可折叠设计节省屏幕空间
- 更符合用户使用习惯

### 二、预览区域优化

**优化前**：
- 使用iPhone 8尺寸（375x667）
- 显示手机状态栏（时间、信号等）
- 预览区较小

**优化后**：
- ✅ 使用iPhone 17尺寸（430x932）
- ✅ 采用H5样式展示，更大气
- ✅ 去掉手机状态栏，只保留页面标题栏
- ✅ 标题栏可配置显示/隐藏、透明/不透明
- ✅ 背景颜色可自定义

**优点**：
- 更大的预览区域，看得更清楚
- H5样式更接近真实小程序效果
- 可配置性更强

### 三、左侧区域优化

**优化前**：
- 只显示"基础组件"
- 无法查看已添加的组件

**优化后**：
- ✅ 增加Tab切换：页面设置 / 已用组件
- ✅ 页面设置Tab：显示可拖拽的基础组件
- ✅ 已用组件Tab：显示已添加到页面的组件列表
- ✅ 点击已用组件可快速定位和选中
- ✅ 已用组件列表显示删除按钮

**优点**：
- 可以快速查看页面包含哪些组件
- 点击组件可快速定位
- 组件管理更方便

### 四、右侧配置面板优化

**优化前**：
- 只有页面名称和页面别名两个配置项

**优化后**：
- ✅ 页面标题：显示在顶部标题栏的文字
- ✅ 页面别名：用于URL路径
- ✅ 分享封面：分享到微信时显示的封面图
- ✅ 页面描述：用于SEO和分享描述
- ✅ 背景颜色：页面的背景颜色（支持颜色选择器）
- ✅ 返回操作：是否显示返回按钮（开关）
- ✅ 透明标题栏：标题栏是否透明（开关）
- ✅ 顶部标题栏：是否显示顶部标题栏（开关）

**优点**：
- 配置项更丰富
- 支持更多页面样式定制
- 实时预览配置效果

## 详细功能说明

### 1. 页面设置Tab

**位置**：左上角Tab切换

**功能**：
- 显示4种基础组件（商品、商品分组、商品搜索、图片广告）
- 支持拖拽组件到预览区
- 组件样式优化，更易识别

### 2. 已用组件Tab

**位置**：左上角Tab切换

**功能**：
- 显示已添加到页面的所有组件
- 每个组件显示类型名称和删除按钮
- 点击组件可选中并在预览区高亮
- 自动滚动到组件位置

**交互**：
- 点击组件 → 选中组件 → 右侧显示配置面板
- 点击删除 → 确认 → 删除组件

### 3. H5预览区

**尺寸**：430x932（iPhone 17）

**结构**：
- 顶部标题栏（44px）
  - 左侧：返回按钮（可配置显示/隐藏）
  - 中间：页面标题
  - 背景：可配置颜色和透明度
- 内容区域
  - 背景：可配置颜色
  - 显示所有组件

**样式**：
- 圆角边框（40px）
- 阴影效果
- 白色背景
- 滚动条

### 4. 页面配置项

#### 页面标题
- **类型**：文本输入框
- **说明**：显示在顶部标题栏的文字
- **验证**：必填
- **实时预览**：修改后标题栏立即更新

#### 页面别名
- **类型**：文本输入框
- **说明**：用于URL路径，如：home、goods-list
- **验证**：必填，只能包含小写字母、数字和连字符
- **提示**：显示格式说明

#### 分享封面
- **类型**：文本输入框
- **说明**：分享到微信时显示的封面图
- **格式**：图片URL
- **提示**：说明用途

#### 页面描述
- **类型**：多行文本框（3行）
- **说明**：用于SEO和分享描述
- **提示**：说明用途

#### 背景颜色
- **类型**：颜色选择器 + 文本输入框
- **说明**：页面的背景颜色
- **默认值**：#ffffff
- **实时预览**：修改后预览区立即更新
- **交互**：
  - 点击颜色预览块打开颜色选择器
  - 也可以直接输入颜色值

#### 返回操作
- **类型**：开关（Switch）
- **说明**：是否显示返回按钮
- **默认值**：开启
- **实时预览**：修改后标题栏立即更新
- **状态显示**：显示当前状态文字

#### 透明标题栏
- **类型**：开关（Switch）
- **说明**：标题栏是否透明
- **默认值**：关闭
- **适用场景**：顶部有大图的页面
- **实时预览**：修改后标题栏立即更新
- **提示**：说明适用场景

#### 顶部标题栏
- **类型**：开关（Switch）
- **说明**：是否显示顶部标题栏
- **默认值**：开启
- **效果**：隐藏后页面全屏显示
- **实时预览**：修改后标题栏立即更新
- **提示**：说明效果

### 5. 产品说明文档面板

**位置**：编辑器右侧，可折叠

**功能**：
- 显示页面编辑器使用说明
- 包含基础操作、页面设置说明、组件说明、注意事项
- 可折叠/展开
- 状态保存到sessionStorage

**交互**：
- 点击按钮切换展开/收起
- 图标变化：◀（展开）/ ▶（收起）
- 动画过渡

## 技术实现

### 1. Tab切换
```javascript
function switchLeftTab(tabName) {
    // 更新Tab状态
    document.querySelectorAll('.left-tab').forEach(tab => {
        tab.classList.remove('active');
    });
    document.querySelector(`.left-tab[data-tab="${tabName}"]`).classList.add('active');
    
    // 更新内容显示
    document.querySelectorAll('.left-tab-content').forEach(content => {
        content.classList.remove('active');
    });
    
    if (tabName === 'page-settings') {
        document.getElementById('page-settings-content').classList.add('active');
    } else if (tabName === 'components') {
        document.getElementById('components-content').classList.add('active');
    }
}
```

### 2. 实时预览更新
```javascript
function updatePageConfig(key, value) {
    const pageData = JSON.parse(sessionStorage.getItem('editingPage'));
    
    if (key === 'name' || key === 'alias') {
        pageData[key] = value;
    } else {
        if (!pageData.config) {
            pageData.config = {};
        }
        pageData.config[key] = value;
    }
    
    sessionStorage.setItem('editingPage', JSON.stringify(pageData));
    
    // 实时更新预览
    updatePreview();
}
```

## 使用说明

### 访问编辑器
1. 打开MG管理后台：http://localhost:8000/dist/mg/index.html
2. 导航到：运营管理 → 商城装修 → 内容管理
3. 点击"创建页面"

### 使用流程
1. 在左侧"页面设置"Tab拖拽组件到预览区
2. 切换到"已用组件"Tab查看已添加的组件
3. 点击组件进行配置
4. 在右侧配置面板设置页面属性
5. 实时查看预览效果
6. 点击右侧文档按钮查看使用说明
7. 保存草稿或发布页面

# 店商供应链管理系统 - 原型演示

## 🎯 项目简介

这是一个店商供应链管理系统的前端原型演示，包含MG管理后台和Lite商户后台两个系统。

- **MG管理后台**：供应链平台管理端，管理商户入驻、商品、订单、财务等
- **Lite商户后台**：商户管理端，商户申请入驻、查看商品、管理订单等

## 🚀 快速开始

### 方法1：统一入口（推荐）

```bash
# 直接打开统一入口页面
open prototype/index.html

# 或使用本地服务器
cd prototype
./start-server.sh
# 访问 http://localhost:8000/index.html
```

**优势：**
- ✅ 在同一个页面中切换MG和Lite系统
- ✅ 无需打开多个浏览器标签
- ✅ 顶部快速切换按钮
- ✅ 支持刷新和重置数据
- ✅ 自动记忆上次访问的系统

### 方法2：直接访问（传统方式）

```bash
# MG管理后台
open prototype/mg/index.html

# Lite商户后台
open prototype/lite/index.html
```

## 🧪 测试指南

完整的测试流程和验证点请查看：[测试指南.md](./测试指南.md)

### 快速测试流程

1. **商品组合创建**
   - 进入「商品管理」→「商品组合」
   - 点击「+ 创建商品组合」
   - 选择供应商和产品
   - 设置价格并创建

2. **商品价格设置**
   - 在商品详情中点击「价格列表」Tab
   - 点击「+ 添加价格」
   - 设置分成比例和时间
   - 查看分成预览并保存

3. **不可发货地区设置**
   - 在商品详情中点击「不可发货地区」Tab
   - 选择地区或应用模板
   - 保存设置

4. **包邮设置**
   - 在商品详情中点击「包邮设置」Tab
   - 选择包邮类型
   - 应用邮费模板

## 💡 业务规则

### 商品组合规则
- 只能组合同一供应商的产品
- 至少选择一个产品
- 销售价格不能为空

### 价格设置规则
- 三方分成比例之和必须等于100%
- 结束时间必须大于开始时间
- 品牌总部分成占商家收益的百分比

### 分成计算规则
```
假设商品售价1000元，手续费率0.6%：
手续费 = 1000 × 0.6% = 6元
可分配金额 = 1000 - 6 = 994元

商家10% + 供应链70% + 私域运营20% = 100%
- 商家收益 = 994 × 10% = 99.4元
- 品牌总部 = 99.4 × 10% = 9.94元
- 商家实际 = 89.46元
- 私域运营 = 198.8元
- 供应链平台 = 695.8元
```

### 数据属性设计

为每个表格行添加 `data-*` 属性支持筛选：

```html
<tr data-app-id="APP001"
    data-status="pending"
    data-subaccount="opened"
    data-scope="compliant"
    data-apply-time="2025-01-10 10:30:00"
    data-audit-time="2025-01-11 14:00:00"
    data-search="YZF001 商户名称 公司名称">
```

### 筛选逻辑

```javascript
// 多条件组合筛选（AND逻辑）
- 时间范围筛选：比较日期字符串
- 状态筛选：匹配 data-* 属性
- 搜索筛选：关键词匹配
- 显示/隐藏：设置 display 样式
```

### 经营范围判断逻辑

```javascript
const scopeCompliant = merchant?.scope && (
    merchant.scope.includes('化妆品') || 
    merchant.scope.includes('日用品') ||
    merchant.scope.includes('美容')
);
```

### 快速测试

**筛选测试**:
1. 选择"伊智付分账状态"为"已开通"
2. 选择"营业执照经营范围"为"符合"

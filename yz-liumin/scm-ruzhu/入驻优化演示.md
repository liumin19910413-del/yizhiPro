## 💡 业务逻辑

### 单店申请逻辑

```javascript
1. 筛选单店商户号（!brandId）
2. 用户选择商户号
3. 填写申请人信息
4. 提交申请
5. 创建申请记录（brandMerchantId: null, brandRatio: 0）
```

### 连锁申请逻辑

```javascript
1. 筛选连锁商户号（有brandId）
2. 用户多选商户号
3. 为每个商户号设置：
   - 总部是否分成
   - 分账接收方（如果分成）
4. 填写申请人信息
5. 提交申请
6. 为每个商户号创建独立的申请记录
```

### 总部分成逻辑

```javascript
if (总部是否分成 === '是') {
  // 显示分账接收方下拉列表
  // 必须选择分账接收方
  // 创建申请时包含：
  // - brandMerchantId: 品牌总部商户号
  // - brandRatio: 10（默认）
  // - profitReceiverId: 选择的接收方ID
  // - contracts: 包含brand类型的协议
} else {
  // 不显示分账接收方
  // 创建申请时：
  // - brandMerchantId: null
  // - brandRatio: 0
  // - profitReceiverId: null
}
```

## 🎓 使用建议

### 单店商户

1. 选择"单店"类型
2. 选择您的商户号
3. 填写申请人信息
4. 查看经营范围提示
5. 确认无误后提交

### 连锁商户

1. 选择"连锁"类型
2. 勾选需要申请的商户号（可多选）
3. 为每个商户号设置是否需要总部分成
4. 如需分成，选择分账接收方
5. 填写申请人信息
6. 查看经营范围提示
7. 确认无误后提交

## 📝 经营范围说明

### 为什么需要这些经营范围？

供应链平台提供的商品涵盖多个品类，商户的营业执照经营范围必须包含相应的类目才能合法销售。

### 主要经营范围

1. **日用品类**：日常生活用品、清洁用品
2. **纺织品类**：毛巾、被子、家纺
3. **卫生用品**：纸巾、卫生巾、个人护理
4. **化学产品**：洗涤用品、清洁剂
5. **化妆品类**：彩妆、护肤品
6. **母婴用品**：婴儿用品、儿童用品
7. **食品类**：预包装食品、保健品
8. **生鲜类**：新鲜水果

### 如何变更经营范围？

1. 前往当地工商局
2. 提交经营范围变更申请
3. 提供相关材料
4. 等待审批
5. 领取新的营业执照

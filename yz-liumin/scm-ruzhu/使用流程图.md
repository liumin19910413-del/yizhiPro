## 🎮 操作说明

### 步骤1：访问统一入口
```
浏览器地址栏输入：
http://localhost:8000/index-switch.html

看到：
- 渐变紫色背景
- 两个白色卡片
- 鼠标悬停时卡片上浮
```

### 步骤2：选择系统
```
点击左侧卡片：
→ 进入 Lite商户后台

点击右侧卡片：
→ 进入 MG管理后台
```

### 步骤3：使用系统
```
在系统中：
- 点击左侧菜单切换功能
- 使用各种功能
- 查看和管理数据
```

### 步骤4：返回首页
```
点击侧边栏顶部的：
[← 返回系统选择] 按钮

→ 返回统一入口页面
→ 可以选择另一个系统
```

## 🔄 浏览器导航支持

```
浏览器后退按钮：
统一入口 → MG系统 → [后退] → 统一入口

浏览器前进按钮：
统一入口 → [前进] → MG系统

浏览器历史记录：
- 统一入口
- MG系统
- 统一入口
- Lite系统
- 统一入口
```

## 📊 数据同步说明

```
┌──────────────┐
│ localStorage │ ← 数据存储位置
└──────┬───────┘
       │
       ├─→ MG系统读写
       │
       └─→ Lite系统读写

两个系统共享同一个localStorage
数据实时同步，无需手动刷新
```

## ⚡ 性能说明

```
首次加载：
统一入口 → 加载时间：< 100ms

系统切换：
点击卡片 → 页面跳转 → 系统加载：< 200ms

返回首页：
点击按钮 → 页面跳转 → 首页加载：< 100ms

总体体验：流畅快速 ✅
```

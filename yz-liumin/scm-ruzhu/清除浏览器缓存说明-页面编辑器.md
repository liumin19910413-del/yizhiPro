# 清除浏览器缓存说明 - 页面编辑器更新

## 问题说明

如果您在访问页面编辑器时没有看到最新的更新内容（如左侧Tab切换、iPhone 17预览区、8个配置项等），这是因为浏览器缓存了旧版本的JavaScript文件。

## 解决方案

### 方法一：强制刷新（推荐）

**Mac系统**：
- Chrome/Edge: `Command + Shift + R`
- Safari: `Command + Option + R`
- Firefox: `Command + Shift + R`

**Windows系统**：
- Chrome/Edge: `Ctrl + Shift + R` 或 `Ctrl + F5`
- Firefox: `Ctrl + Shift + R` 或 `Ctrl + F5`

### 方法二：清除浏览器缓存

#### Chrome/Edge浏览器
1. 打开开发者工具：`F12` 或 `Command + Option + I` (Mac)
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

或者：
1. 打开设置：`Command + ,` (Mac) 或 `Ctrl + ,` (Windows)
2. 搜索"清除浏览数据"
3. 选择"缓存的图片和文件"
4. 点击"清除数据"

#### Safari浏览器
1. 打开开发菜单：Safari → 偏好设置 → 高级 → 勾选"在菜单栏中显示开发菜单"
2. 点击菜单栏的"开发" → "清空缓存"
3. 刷新页面：`Command + R`

#### Firefox浏览器
1. 打开菜单：`Command + Shift + Delete` (Mac) 或 `Ctrl + Shift + Delete` (Windows)
2. 选择"缓存"
3. 点击"立即清除"

### 方法三：禁用缓存（开发时使用）

**Chrome/Edge开发者工具**：
1. 打开开发者工具：`F12`
2. 打开设置：点击右上角齿轮图标
3. 勾选"Disable cache (while DevTools is open)"
4. 保持开发者工具打开状态

**Safari开发者工具**：
1. 打开开发菜单
2. 勾选"停用缓存"

**Firefox开发者工具**：
1. 打开开发者工具：`F12`
2. 打开设置：点击右上角齿轮图标
3. 勾选"禁用HTTP缓存（当工具箱打开时）"

## 常见问题

### Q: 为什么会有缓存问题？
A: 浏览器为了提高加载速度，会缓存JavaScript、CSS等静态文件。当文件更新后，浏览器可能仍然使用旧的缓存版本。

### Q: 强制刷新和普通刷新有什么区别？
A: 
- 普通刷新（`Command + R`）：浏览器会检查文件是否更新，但可能仍使用缓存
- 强制刷新（`Command + Shift + R`）：浏览器会忽略缓存，重新下载所有文件

### Q: 清除缓存会影响其他网站吗？
A: 
- 如果只清除当前网站的缓存，不会影响其他网站
- 如果清除所有缓存，其他网站的加载速度可能会暂时变慢

### Q: 开发时如何避免缓存问题？
A: 
1. 在开发者工具中禁用缓存
2. 使用无痕模式
3. 在URL后添加时间戳参数（如：`?t=20260129`）

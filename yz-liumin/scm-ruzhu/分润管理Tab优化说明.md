## 🔧 技术实现

### 1. HTML结构
```html
<div class="profit-tabs">
    <div class="profit-tab active" data-tab="brand">
        品牌总部分成配置
    </div>
    <div class="profit-tab" data-tab="distributor">
        经销商单位分成设置
    </div>
</div>

<div class="profit-tab-content">
    <div id="brand-tab-content" class="tab-pane active">
        <!-- 品牌总部分成配置内容 -->
    </div>
    <div id="distributor-tab-content" class="tab-pane">
        <!-- 经销商单位分成设置内容 -->
    </div>
</div>
```

### 2. JavaScript切换逻辑
```javascript
function switchProfitTab(tabName) {
    // 更新Tab导航状态
    document.querySelectorAll('.profit-tab').forEach(tab => {
        tab.classList.remove('active');
    });
    document.querySelector(`.profit-tab[data-tab="${tabName}"]`)
        .classList.add('active');
    
    // 更新Tab内容显示
    document.querySelectorAll('.tab-pane').forEach(pane => {
        pane.classList.remove('active');
    });
    document.getElementById(`${tabName}-tab-content`)
        .classList.add('active');
}
```

### 3. CSS样式
- `.profit-tabs` - Tab导航容器
- `.profit-tab` - 单个Tab按钮
- `.profit-tab.active` - 激活状态的Tab
- `.tab-pane` - Tab内容面板
- `.tab-pane.active` - 显示的Tab内容
- 淡入动画效果

## 💡 设计理念

### 1. 信息层级优化
- 使用Tab页将两类不同的分润配置分开
- 避免页面过长需要滚动
- 用户可以快速定位到需要的信息

### 2. 视觉引导
- 使用图标增强识别度
- 激活状态有明显的视觉反馈
- Badge标签让数据更醒目

### 3. 交互流畅
- 点击即可切换，无需等待
- 淡入动画让切换更自然
- 保持原有功能不变
